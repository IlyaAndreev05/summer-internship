GPSS (General Purpose Simulation System) — язык имитационного моделирования.

Транзакт — динамический объект GPSS, который перемещается по блокам модели.
Транзакты создаются блоком GENERATE и уничтожаются блоком TERMINATE.

Блок GENERATE создаёт транзакты через заданные интервалы времени.
Синтаксис: GENERATE A,B,C,D,E
A — средний интервал между транзактами
B — разброс интервала
C — время появления первого транзакта
D — максимальное число транзактов
E — приоритет

Блок TERMINATE уничтожает транзакт и уменьшает счётчик завершения на 1.
Синтаксис: TERMINATE A
A — величина уменьшения счётчика завершения

Блок ADVANCE задерживает транзакт на заданное время.
Синтаксис: ADVANCE A,B
A — среднее время задержки
B — разброс времени задержки

Блок QUEUE — сбор статистики ожидания.
Синтаксис: QUEUE A,B
A — имя очереди
B — число занимаемых мест

Блок SEIZE занимает устройство.
Синтаксис: SEIZE A
A — имя устройства

Блок RELEASE освобождает устройство.
Синтаксис: RELEASE A
A — имя устройства

Блок TRANSFER направляет транзакт по заданному маршруту.
Синтаксис: TRANSFER A,B,C
Режимы: безусловный, статистический, по условию.
